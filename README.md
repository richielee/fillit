# fillit
fillit

Given an array of Tetrominos, the programme will return the smallest square that would contain all the Tetrominos. 

Example input:
....$\n
##..$\n
.#..$\n
.#..$\n
$
....$\n
####$\n
....$\n
....$\n
$
#...$\n
###.$\n
....$\n
....$\n
$
....$ 
##..$ 
.##.$ 
....$ 

Output: 
DDAA$ 
CDDA$ 
CCCA$ 
BBBB$ 

Where 'A', 'B', 'C' and 'D' represents the order of the Tetrominos.
